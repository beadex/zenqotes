# ZenQotes

## Usage:

`zenqotes <cmd>`

Commands:

`zenqotes`        Get a random quote                                   [default]

`zenqotes today`  Get a quote for today

Options:

`--version`  Show version number                                       [boolean]

`--help`     Show help                                                 [boolean]

## Install & Use

`npm install -g zenqotes`

Or use it on the go:

`npx zenqotes`
